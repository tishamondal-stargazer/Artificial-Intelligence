# Theory – Tic Tac Toe Game

Tic-Tac-Toe is a simple and well known two-player game played on a 3x3 grid. The game is typically played using two symbols: X and O.

The objective of the game is to place three identical symbols in a row, column, or diagonal before the opponent does. Players take turns placing their symbols in empty cells until either a player wins or the grid becomes full.

In Artificial Intelligence, Tic-Tac-Toe is often used to demonstrate game search techniques and decision-making strategies. The game state can be represented as a grid where each cell may contain X, O, or be empty.

The main components of the Tic-Tac-Toe game include:

• Game board representation  
• Player moves and turn switching  
• Checking winning conditions  
• Detecting draw situations  

The winning condition occurs when a player successfully places three of their marks in a straight line either horizontally, vertically, or diagonally.

This practical demonstrates how Python can be used to implement a game engine that manages the board, processes player input, and determines the outcome of the game.
