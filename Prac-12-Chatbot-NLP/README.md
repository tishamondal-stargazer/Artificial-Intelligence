Practical 12 – AI Chatbot using NLP

Aim  
Build an AI chatbot for FAQs using NLP.

Description  
This chatbot uses:
• Bag of Words
• Cosine Similarity

It matches user input with predefined questions.

Tools Used  
Python  
NLTK  
Scikit-learn  

Run  

python ai_Prac12.py

Example  

User: What is AI?  
Bot: AI stands for Artificial Intelligence.
