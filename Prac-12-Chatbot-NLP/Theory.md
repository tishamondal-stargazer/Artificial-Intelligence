A chatbot is an AI-based system that interacts with users using natural language. It uses Natural Language Processing (NLP) techniques to understand and respond to user queries.

Basic chatbot working includes:

Text preprocessing (tokenization, stopword removal)
Feature extraction (Bag of Words / TF-IDF)
Similarity matching (Cosine Similarity)

The chatbot compares user input with predefined data and returns the most relevant response.

Applications:

Customer support
Virtual assistants
FAQ systems
