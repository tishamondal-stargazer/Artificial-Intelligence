Practical 09 – Bayes Theorem for Rain Prediction

Aim  
Evaluate Bayes Rule for rain prediction from given scenario.

Description  
This practical demonstrates the application of Bayes Theorem to calculate conditional probability.

Given:
• Probability of cloudy weather  
• Probability of rain  
• Probability of cloudy given rain  

Using Bayes Theorem, we compute:
P(rain | cloudy)

This helps in predicting rain based on observed weather conditions.

Tools Used  
Python  
VS Code  

How to Run  

python ai_Prac9.py

Sample Output  

P(cloudy) = 0.4  
P(rain) = 0.2  
P(cloudy | rain) = 0.85  

P(rain | cloudy) = 0.425  
Probability = 42.5%
