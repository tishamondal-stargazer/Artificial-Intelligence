Prelab Practical 01 – Magic Square Implementation

Aim:
To write a Python program to implement a Magic Square of order n.

Objective:
• To understand the concept of Magic Squares.
• To develop logical thinking and puzzle solving capability.

Description:
A Magic Square is a square matrix of order n × n containing distinct integers from 1 to n² arranged in such a way that the sum of the numbers in each row, each column and both diagonals are equal.

This practical demonstrates the implementation of the classical Siamese algorithm used for constructing odd ordered magic squares. The program accepts an odd number as input and generates a valid magic square while calculating the corresponding magic constant.

The implementation highlights algorithmic reasoning, matrix operations and logical problem solving using Python.
