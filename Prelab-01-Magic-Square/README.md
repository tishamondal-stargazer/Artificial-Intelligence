# Prelab 01 – Magic Square Problem

## Aim
To generate a Magic Square using Python and understand the concept of number arrangement in Artificial Intelligence problems.

## Description
A Magic Square is a square grid of numbers where the sum of numbers in every row, column and diagonal is the same. The concept of Magic Squares is used in Artificial Intelligence and mathematics to demonstrate structured problem solving and logical arrangement of numbers.

In this practical, a Python program is implemented to generate a Magic Square of a given size. The program arranges numbers in such a way that the sum of each row, column and diagonal remains constant.

Magic Squares are commonly used in algorithm design problems to demonstrate systematic placement of numbers and rule based computation.

## How to Run the Program

1. Install Python on your system.

2. Clone or download the repository.

3. Navigate to the folder:

Artificial-Intelligence/Prelab-01-Magic-Square

4. Run the program:

python ai_Prac1_Prelab.py

5. Enter the size of the magic square when prompted.
