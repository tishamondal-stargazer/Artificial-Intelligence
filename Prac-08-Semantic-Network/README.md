Practical 08 – Semantic Network using Predicate Logic

Aim  
Demonstrate the concept of Semantic Network using predicate logic to represent knowledge.

Description  
This practical demonstrates the implementation of a Semantic Network, a knowledge representation technique in Artificial Intelligence.

A semantic network represents knowledge using:
• Nodes → Objects or Concepts  
• Edges → Relationships between them  

In this program:
• "is_a" represents ISA relationship (inheritance)
• "can" represents abilities

The program builds a knowledge base and performs inference using inheritance.

Example:
• Canary is a Bird  
• Bird is an Animal  
• Bird can Fly  
• Tweety is a Canary  

The system infers that:
• Tweety is a Bird  
• Tweety is an Animal  
• Tweety can Fly
