Linear Regression is a supervised machine learning algorithm used to predict continuous values by establishing a relationship between independent and dependent variables.

It assumes a linear relationship of the form:
y = mx + c

Where:

y = dependent variable
x = independent variable
m = slope
c = intercept

The model is trained by minimizing the error between predicted and actual values using methods like Least Squares.

Applications include:

Sales prediction
Trend analysis
Forecasting
